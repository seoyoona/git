Css Display

table-cell

= 문법과 상관 X css로 모양만 변경 (td 역할)

+ Property

  • display : table-cell
  • vertical-align : midddle
  • height

hidden

= 요소 숨김

+ Property

  • display : none; = 숨김
  • color : rgba(@,0,0,0); = opacity 0으로 숨김
    • visibility : hidden;
      → visibility : visible;
    • opacity :0;
      → opacity : 1;
    • color : transparant;
      → color : @;

overflow

⇒ html{overflow-y : scroll;} = 초기화 코드에 들어가야 layout 안흔들림
  • auto : 넘치는 쪽 자동 스크롤 생성
  • scroll : 강제 스크롤 생성
  • visible : 넘치는 상태 그대로 보임
  • hidden : 넘치는 부분 자름
  • overflow-x : x축
  • overflow-y : y축
