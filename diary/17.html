CSS Font

Fonts

+ Property

- font-family : 글씨체
  • serif (명조체) : 시스템 글꼴 중 아무거나
  • sans-serif (고딕체) : 시스템 글꼴 중 아무거나 (mac / window 호환에서 특히 중요)
- font-size : 글씨체 크기
  • em : 16px (부모에 상대적)
  • rem : root에 상대적
  • vw : viewport width / vh : viewport height
- font-style : 폰트 기울기 etc. (ex. italic)
- font-virant : 대소문자
  • small-caps : 소문자 크기 대문자
- font-weight : 글씨체 두께
- line-height : 행간
  • 단위 x : ~배 (유일하게 적용) (ex. 150% = 1.5)

+ Font short-hand (선언 순서)

= font: [font weight, font-style, font-virant] [font-size/line-height] [font-family]
