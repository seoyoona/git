가상 선택자

= 하이퍼링크에 스타일 부여 (둘 이상 선택시 순서 중요)
  • a:link = 링크 걸린 상태
  • tag:visited = 방문한 링크
  • tag:hover = mouse on
  • tag:active = 왼쪽 마우스 누르고 있을 시 (클릭하고 있을 떼)
  • tag:focus = 입력 상태에 커서가 있을 떼 (탭으로 선택시)
  • tabindex="-1" : tab 건너뜀 (중요도 X)
  • tabindex="0" : tab O (default)
  • tabindex-="1" : tab O (우선순위)

가상요소 선택자

  • ::first-letter = 첫번째 글자
  • ::first-child = 첫 요소
  • ::last-child = 마지막 요소
  • :before = 태그 안에 자식으로 추가(앞에, 메뉴에 많고 액티비티 효과 많음)
  • :after = 뒤에 추가
  • nth-child(n) = 부모 모든 자식 요소 중 몇 번째
    • tag가 뭐든 그냥 하위요소 중 하나
    • .box>:nth-child(n){}
  • nth-type-of(n) = 하위 중 특정 태그만
    • .box>span:nth-of-type(n){}
