* HTML basic elements *

TEXT

  • blockquote : 인용문 (들여쓰기, 스크린리더기에서 인용문처럼 읽어줌)
  • em : italic (스크린리더기에서 강조)
  • i : italic (단순 화면상)
  • address : italic (주소로 인식)
  • strong : bold (스크린리더기에서 강조)
  • b : bold (단순 화면상)
  • mark : 형광펜
  • small : 글씨 작게 (주의사항, 법적제한, 저작권 etc.)
  • sub : 아래첨자
  • sup : 윗첨자
  • hr : 수평선

LINK

  • a<section<h2+p
  • <a href="#content"~</a> = content 누르면 #content로
  • <p id="content"~</p> = content 누르면 #content로
  • <title> = 문서제목
  • <title=""> = 태그의 제목
    • 스크린 리더기가 읽어줌
    • li마우스 올리면 도움말 기능
    • target 속성 사용시, title로 "새창 열림" 알려야함
  • <target="_blank"> = open in new window
  • alt(img) = alternative text
    • 접근성 향상 (스크린리더기)
    • SEO
    • 이미지 안 뜰 경우 대체 택스트
  • 블록 = 자기 공간 O
  • 인라인 = 자기 공간 X + 나란히 배치
  • 이미지 = 자기 공간 O, but 나란히 배치

LIST

  • ol = order list (순서O 목록)
  • li = 개별 항목
  • ul = unorder list (순서X 목록)

DESCRIPTION

  • dl = description list
  • dt = description title
  • dd = hyper text markup language
EX.<de>
           <dt>html</dt>
           <dd>description</dd>
     </de>
