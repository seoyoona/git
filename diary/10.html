Table & Form

Table

= table>tr>td
  • table : 표
  • tr : table row (열)
  • td : table data (행)
  • <thead> : 표 머리말
  • <tbody> : 표 본문
  • <tfoot> : 표 꼬리말
⇒ for 기계 이해 / 화면상 X

<style>
     {border:1px solid;}
     td{border:1px solid;}
</style>
  • border : 윤곽선
  • 1px : 선 두께
  • solid : 선 모양(직선)

  • <caption> : table title
  • <colgroup> : 열 그룹(칸)
  • <col> : column group

<colgroup>
     <col class="c1"
     <col class="c2"
</colgroup>

<style>
     .c1{background-color : yellow;}
</style>

⇒ table<colgroup<col
⇒ table<tr(<th)<td
  • colspan : 셀병합 (열)
    • → <td colspan="4">
    • - 남아있는 td 지우기
  • rowspan : 셀병합 (행)
    • → <td rowspan="2"
    • - 밑 열 해당 td 지우기
    - border-collapse : "collapse"
  • scope="col" → 스크린리더기 (열)
  • scope="row" → 스크린리더기 (행)

Form

= 사용자와 대화 가능
= 사용자 의견 제출 etc. 서비스 제공 이외 가능 O
  • form : 입력 양식 태그
    • action. data = 전송할 서버 주소
    • method.data = 전송 방법 (get, post)
  • fieldset : form 요소 그루핑 (같은 형태의 서비스)
  • legend : fieldset title

<form action="서버주소" method="get">
     <fieldset>
          <legend>제목</legend>
          내용 <input type="txt">
          내용 <input>="password">
     </fieldset>
</fieldset>

+ method

  • get : 사용자 입력값 주소창에 노출 O
  • post : 사용자 입력값 주소창에 노출 X

+ input/type

  • 아이디 : text
  • 비밀번호 : password
  • 검색 : search
  • 이메일 : email
  • 전화번호 :tel
  • 컬러 : color
  • 제출 : submit
  • 췻소 : reset
  • <button>클릭</button>
  • 이미지 : img
  • 파일 전송 : file
