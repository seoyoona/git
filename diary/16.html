  2. head 요소 내 style 요소
  3. head 요소 내 style 요소 내 @import 문
  4. <link> 로 연결된 css 파일
  5. <link> 로 연결된 css 파일 내 @import 문

명시도

  1. !imoportant : 최우선
  2. inline style : 1000
  3. ID : 100
  4. Class : 10
  5. 속성 선택자 : 10
  6. 가상 클래스 속성 선택자 : 10
  7. 태그 선택자 : 1
  8. 전체 선택자 : 0

+ style

+ 상속관계
