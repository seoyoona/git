HTML "Form"

input / type

  • value = input 창에 담기는 값, 사용자 데이터 값
  • id = 디자인용 식별자
    • 중복 허용 X
    • <style>#usertext{~}
  • name = 백단용 식별자
    • 데이터가 저장디는 그룹명
    • DB에 저장되는 필드값

textarea / select

  • textarea : 텍스트 입력 가능 공간
    • <p>내용:<br><textarea name="" id="" cols="" row="">
  • select : 선택지 중 고르기
    • <p>내용:<select name="" id="">
    •                <option value>="받는 데이터값"내용
    •           </select>
    •           <input type="tel" name="tel2" id="tel2" placeholder(또는 value)="">
    • <p>
  • Example Practice

label

= 스크린리더기가 input에 아이디 입력하라고 알려줌
  • title = input의 title

CSS "Selector"

  • id selector : #id{property}
    • 중복 사용 불가능
  • class selector : .class{property}
    • 중복사용 가능
  • asterisk : 전체 선택자
    • *{margin:0; padding:0; list-style:none;} = 초기화모드
  • 자손 선택자 : 상위태그 하위태그{property}
  • tag selector : tag{property}
