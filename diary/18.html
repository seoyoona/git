CSS Color

color

- image background = {background:url(파일 경로);}
- 표시 방법
  • 관용색 : color name
  • hex code : rgb 값 16진수 표현 (#333)
  • rgb : (red,green,blue) 각각의 포화도 입력
    • ex. (163,229,77)
  • hsl : (hue,saturation,lightness)
    • (0~360, 0~100%, 1~100%)
  • rgba/hsla : alpha 채널 추가 (opacity)
    • ex. rgb(150,229,77,0.8) = opacity애서 0 생략 가능 (.8)
- padding 값 높아지면 box 크기 커짐

word-spacing / letter-spacing

  • letter-spacing : 자간
  • word-spacing : 어간

text-decoration / text-transform

= text-decoration (밑줄 속성)
  • none : 밑줄x (링크)
  • underline : 밑줄
  • overline : 위에 줄
  • line-through : 취소선
= text-transform
  • uppercase : 대문자
  • lowercase : 소문자
  • capitalize : 앞에만 대문자

text-shadow

= text-shadow : x y blur color;
  • 콤마 사용 x
  • 콤마 사용 하는 경우 : () 안에 인자 사용시

Paragraph

text-align : 문단 정렬

= 여백 확보 필수!!!
= block에 적용시 하위 인라인 정렬
  • left : 왼쪽 정렬
  • right : 오른쪽 정렬
  • center : 가운데 정렬
  • justify : 양쪽 정렬
- margin : ~px (상하) ~px (좌우)
  • margin 병합 : 한 쪽 방향만 몰아서 줌 (위아래만 적용, 좌우에 적용 x)

vertical-align

= 수직정렬 (inline / table-cell box(td)) : 인라인끼리
.class img{vertical-align:~}

text-indent

= 들여쓰기
  • - (마이너스) : 내여쓰기 (ex. -9999...px : text 없어짐)

word-wrap / word-break

  • word-wrap : 영문 줄바꿈
  • word-break : 모든 언어 줄바꿈 (CJK = 글자기준 / en : 단어 기준)

white-space

= 공백처리
  • normal
  • nowrap : 줄바꿈 x
  • pre : 작성한대로 입력
  • pre-wrap : 줄바꿈 유지 + 작성한 입력값
  • pre-line : 줄바꿈 유지 + 작성한 줄바꿈만 + 공백x

text-overflow

= 말줄임표 (너비제한 + 줄바꿈 x + 넘치는거 자름 + 줄임표)
= 블럭 요소에 적용
  • width
  • overflow : hidden /scroll
  • text-overflow : ellipsis (말줄임표)
  • white-space : nowrap (무조건 한줄로 결과값)
** a{display:block} = a를 블럭요소로 만듬

display

= css적을 화면 가시성 조정
  • inline : 인라인 성격 (행으로 정렬)
  • inline-block : 나란히 블록
