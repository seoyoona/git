* html5 basic structure *

Structure

Charset

<meta charset"UTF-8"> = 문자의 문자 세트 생성

제목요소

content's heading = h1-h6 (h$*6)

+ Entity codes

  • a<b = &lt;
  • b>a = &gt;
  • &nbsp; = 한 칸 띄기
  • &#8361 = 원화 표시
  • &rarr; = 오른쪽 화살표
  • &larr; = 왼쪽 화살표
  • &rArr; = 오른쪽 두줄 화살표
  • &lArr; = 왼쪽 두줄 화살표

+ codes

  • <strong> = 글자크기 동일+볼드
  • <span> = 인라인 (한줄로)
  • <h1+p> = 형제관계 생성
  • <br> = p 요소 안에서 줄바꿈, p/p 거리>p/br 거리
  • <pre> = 쓴 그대로 보임
  • <address> = 주소
    • <a href="mailto:~"~</a>
    • <a href="tel:~"~</a>
    • <a href="sms:~"~</a>

+ keyboard shorcuts

  • ctrl+k/s = 단축키 설정
  • ctrl+alt+b = 라이브 서버
  • ctrl+shift+a = wrap
  • shift + Home = 블럭 선택
  • ctrl+alt+방향키 = 열 편집
