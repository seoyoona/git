git 로컬 작업 순서

  • 폴더별 최초 한 번 깃 초기화
    • git init
  • 스테이지에 올림
    • git add .
  • pc별 최초 한 번 사용자 등록
    • git config --global user.email ""
    • git config --global user.name ""
  • 폴더별 최초 한 번 브랜치명 변경
    • git branch -M main

git 원격 작업 순서

  • 폴더별 최초 한 번
    • create repository : 버트 클릭 후 정보 입력해 저장소 생성
  • 생산된 repository 주소 복사
  • git add . : 편집파일 스테이지로 올림
  • git commit -m "commit message" : 버전 생성
  • git push origin main : 원격 저장소에 코드 올림
